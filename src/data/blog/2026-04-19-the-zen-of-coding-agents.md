> "LLM output is a very useful set of random words."

My friend [Kao Félix](https://kaofelix.dev/) always says something like that.

Like all zen ideas, this one is hard to see.

LLM output looks like real talk. It triggers your deepest emotions and fears. It is incredibly useful.

But LLM output follows no philosophy, mental model, cultural norms, ethics, reason or even logic. **It is a process**. And it's a process that we don't fully understand. We don't even know if the process is, by its nature, understandable.

Agentic output contains all **combined knowledge of humankind**, but it's also nothing.

It doesn't get any more Zen than that.
